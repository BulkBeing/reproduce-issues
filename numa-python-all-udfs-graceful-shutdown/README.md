# Implementing graceful shutdown for all Numaflow Python UDFs

For testing the changes in https://github.com/numaproj/numaflow-python/pull/337

This directory should be within `packages/pynumaflow/examples/` to build the UDF image as it depends on `pynumaflow` by path. All UDFs are implemented in this codebase. The pipeline runs all UDFs.
